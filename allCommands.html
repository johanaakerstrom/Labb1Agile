# Commands Description
1 cd .. Backa ett steg i mappen
2 cd C/Kod/Labb1 Gå till en specifik mapp
3 ls Listar alla mappar/filer
4 git remote add origin Länken till repository så att den kopplar upp sig till repot
5 git init Initierar ett tomt repository
6 git branch development Skapar en branch som heter "development"
7 git checkout development Byter till "development" branchen
8 touch .gitignore Skapar en "git ignore" fil
9 git status Kollar vad som ligger i staging
10 git add . Lägger till alla nya ändringar i staging
11 git pull Hämtar det senaste repot
12 git push Pushar upp till repot
13 git checkout -b MagnusBranch Skapar en branch som heter MagnusBranch och byter till den
14 git commit -m "Merge" Skapar en ny commit med ändringar från staging
15 git add .gitignore Lägg till specifikt .gitignore i staging
16 git log Visar historiken över commits i projektet
18 git tag Skapa, lista eller tar bort taggar för specifika commits
